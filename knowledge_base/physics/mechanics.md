# 高中物理 · 力学

## 基本概念

力学研究物体的运动及其原因。核心量：位移、速度、加速度、力、动量、能量。

## 运动学

### 匀速直线运动
- 位移：$s = vt$
- 速度：$v = s/t$

### 匀变速直线运动
- $v = v_0 + at$
- $s = v_0 t + \frac{1}{2}at^2$
- $v^2 - v_0^2 = 2as$

### 自由落体
- $v = gt$
- $h = \frac{1}{2}gt^2$

## 力学三大定律

### 牛顿第一定律（惯性定律）
一切物体在没有受到力或合力为零时，保持静止或匀速直线运动。

### 牛顿第二定律
$F = ma$，加速度与合外力成正比，与质量成反比。

### 牛顿第三定律
作用力与反作用力大小相等、方向相反、作用在同一直线上。

## 常见易错点

1. 混淆速度与加速度方向（加速时同向，减速时反向）
2. 摩擦力方向判断错误（与相对运动或相对运动趋势方向相反）
3. 整体法与隔离法混淆
4. 向心力与离心力概念混淆
5. 机械能守恒条件判断错误（只有重力或弹力做功）
6. 动量守恒条件判断错误（系统合外力为零）
