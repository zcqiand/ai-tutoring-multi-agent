# 积分

## 定义

积分是导数的逆运算。不定积分求原函数，定积分求曲线下面积。

## 基本积分法则

- ∫ x^n dx = x^(n+1)/(n+1) + C  (n ≠ -1)
- ∫ 1/x dx = ln|x| + C
- ∫ e^x dx = e^x + C
- ∫ sin(x) dx = -cos(x) + C
- ∫ cos(x) dx = sin(x) + C

## 牛顿-莱布尼茨公式

∫[a,b] f(x) dx = F(b) - F(a)，其中 F 是 f 的原函数。

## 常见易错点

1. 忘记加常数 C（不定积分）
2. 换元积分法忘记回代变量
3. 定积分上下限搞反
