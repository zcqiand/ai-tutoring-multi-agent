# 导数

## 定义

导数是微积分的核心概念，描述函数在某一点的瞬时变化率。

几何意义：函数图像在该点的切线斜率。
物理意义：位移对时间的导数是速度，速度对时间的导数是加速度。

## 基本求导法则

- 常数法则：(c)' = 0
- 幂法则：(x^n)' = n·x^(n-1)
- 和法则：(f+g)' = f' + g'
- 积法则：(f·g)' = f'·g + f·g'
- 商法则：(f/g)' = (f'·g - f·g') / g²
- 链式法则：(f(g(x)))' = f'(g(x))·g'(x)

## 常见易错点

1. 链式法则漏乘内层导数——最常见错误
2. 商法则分子符号搞反（应是 f'g - fg'，不是 fg' - f'g）
3. e^x 的导数是 e^x 本身，不是 x·e^(x-1)
